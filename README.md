# Pomodoro Discord Bot
This Discord bot uses the Pomodoro timer to send reminders when studying. Each study session is 25 minutes, with a short break being 5 minutes and a long break (after 4 study sessions) being 15 minutes. In the demo (both video and server channel), each study session is 3 minutes with a 1 minute break in between. Additionally, it has calculator functions to help with math problems.

To see a preview of the commands and (test it out for yourself if I'm online!), join the testing Discord server below:
https://discord.gg/xCzJ27r73y

# Commands
Calculator:
!add - adds two numbers
!sub - subtracts two numbers
!mult - multiplies two numbers
!div - divides two numbers

Study:
!hello - say hi!
!start - starts the study session
!end - ends the study session and shows the total amount of time studied for
Breaks and session counting is automatic
